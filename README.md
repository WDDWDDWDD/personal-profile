# 个人简介页面

这是一个简洁美观的个人简介页面，展示我的技术栈、项目经验和联系方式。

## 功能特点

- ✅ 响应式设计，适配各种设备
- ✅ 现代化的UI界面和动画效果
- ✅ 清晰的信息分类展示
- ✅ 交互式技能标签
- ✅ 优雅的渐变色背景

## 技术栈

- HTML5
- CSS3 (Flexbox, Grid, 渐变)
- JavaScript (ES6+)
- Font Awesome 图标库

## 页面结构

```
personal-profile.html    # 主页面文件
README.md               # 项目说明文档
```

## 如何使用

1. 直接使用浏览器打开 `personal-profile.html` 文件
2. 或者部署到任何静态网站托管服务

## 自定义内容

你可以修改以下部分来个性化页面：

1. **个人信息** - 姓名、简介、标签
2. **技术栈** - 更新技能标签
3. **项目经验** - 添加你的项目
4. **联系方式** - 更新邮箱、GitHub等
5. **样式** - 修改颜色、字体等CSS变量

## 浏览器支持

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+
